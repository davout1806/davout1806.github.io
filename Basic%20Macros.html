Basic Macros

 14th March 2026 at 3:51pm
$:/tags/MacroCampaign: GreyhawkCore Wiki StructDnD RuleMacroPublic

<<include tiddler>> - Include a tiddler as a chapter
<<item-box tiddler emptyMessage>> - List items in a box as a grid
<<list-box tiddler emptyMessage>> - List tiddlers as a grid of links
<<list-folds tiddler emptyMessage>> - List tiddlers as rows of foldable tiddlers
<<list-icons tiddler emptyMessage>> - List tiddlers as a grid of icons
<<list-abilities classes level>> - List class-level abilities as a comma delimited list
<<htabs tiddler>> - List tiddlers in a tab panel
<<vtabs tiddler>> - List tiddlers in a vertical tab panel
<<link tiddler>> - Link to a tiddler, using it's caption if it exists
<<links tiddlers>> - Link to a list of tiddlers, using their captions if it exists
<<p tiddler>> - Include a tiddler as a paragraph
<<p2 tiddlers>> - List tiddlers as a series of paragraphs
<<p3 tiddlers>> - List tiddlers as a series of paragraphs
<<imageuri tiddler>> - URI of image tiddler

Old Macros - To be deprecated
<<h3 tiddler>> - Include a tiddler as a third level chapter
<<hover tiddler>> - Popup tiddler's content
<<snippet title tiddler>> - Include tiddler as a floating snippet
